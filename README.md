Instruções:

Para rodar o projeto, instale os modulos, rodando os comandos
npm install
npm install mysql2 dotenv
npm install express ejs

Abra um compilador de MySQl, e rode os scripts contidos em /scripts/script.sql

Caso seja necessário, troque a senha e usuário em /config/db.js

Após isso, use o comando
npx nodemon server.js

E acesse http://localhost:5000/ no navegador
